# move_old_files.py

> **用于查找指定目录下修改时间小于timestamp的文件，并移动到其他目录之下，并且保持目录结构路径**
>
>  比如 res/test1/test2/aa.png
>  移动到 res2目录下，变成
>  比如 res2/test1/test2/aa.png

使用示例

`python move_old_files.py check-dir output-dir`

其中指定输出路径参数为可选

- 如果有，则将文件移动到指定的移动目录并保持目录结构
- 如果没有，则在与检查目录相同层级的位置创建名为 "检查目录名X" 的默认移动目录，其中X为一个整数



# timestamp_utils.py

> **用于生成指定日期时间的时间戳**

使用示例

`python timestamp_utils datetime`

其中支持的日期时间格式如下：

```shell
支持的格式：
- 2021/1/2-00:00:00
- 2021/1/2 00:00:00
- 2021-1-2-00:00:00
- 2021-1-2 00:00:00
- 2021.01.02-00:00:00
- 2021.01.02 00:00:00
- 2021/01/02  # 只有日期时，默认时间为 00:00:00
- 2021-01-02
- 2021.01.02
```
